Healthapp

This application allows a user to search for local health-oriented businesses based on individual needs and goals. On the calorie counter page a user can track calories consumed with the food list and determine their daily caloric needs with the calorie calculator.

Previous Next

Technical Sheet

This application is responsive to screen sizes across devices. It incorporates Semantic UI utilities including multiple dropdowns, containers, and inputs. It also makes calls to 3 APIs, Yelp, Zomato, and Nutritionix to output information to the user. Finally the application stores and recalls the user's food input from the browser's local storage.
