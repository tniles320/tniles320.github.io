Bindrr

A web application for property management companies and rental agents to organize contact information for leads while filling vacant rental properties.

Technical Sheet

This application uses MySQL and Sequelize to store and manipulate user data, Passport and bcrypt to authenticate users and hash passwords, and handlebars to dynamically display data from MySQL tables.

Technologies

Code technologies used for this project:

  • HTML
  • CSS
  • Javascript
  • jQuery
  • Node
  • Express-handlebars
  • MySQL
  • Sequelize
  • Figma
  • Materialize
